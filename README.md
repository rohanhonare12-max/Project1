# Number Guessing Game

## Description

The Number Guessing Game is a Java console application where the computer generates a random number and the user tries to guess it. The program provides feedback after each guess, indicating whether the guessed number is too high or too low until the correct number is found.

## Features

* Generates a random number automatically
* Accepts user input for guesses
* Provides "Too High" or "Too Low" feedback
* Counts the number of attempts
* Displays a success message when the correct number is guessed
* Input validation for user entries

## Technologies Used

* Java
* Scanner Class
* Random Class
* Loops
* Conditional Statements
* Exception Handling

## How to Run

### Compile the Program

javac NumberGuessingGame.java

### Run the Program

java NumberGuessingGame
